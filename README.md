# Contraptions

A collection of small hand-built programs, each living in its own
directory and each doing one small job. Some run quietly in the
background; some you only notice when something goes wrong. They share
no code, no theme, and no roadmap. Things land here when they are
useful enough to keep but too small to deserve a repository of their
own.

Each directory has its own README describing what its contraption does
and how to run it.
